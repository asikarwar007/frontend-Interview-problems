# Heap Sort

## Description
Implement a function that performs a heap sort.

## Example
```javascript
const arr = [5, 3, 8, 4, 2];
console.log(heapSort(arr)); // [2, 3, 4, 5, 8]
```

## Constraints
- The input array can contain any number of elements.

## Notes
This question tests your understanding of sorting algorithms.
